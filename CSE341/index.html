General Information

This course offers an overview of database systems, including the essentials of database design, modeling, and architecture with practical applications. The course begins with foundational concepts including entity-relationship models, the relational model, and SQL.

At the end of this course, the student should acquire the skills necessary to design, implement, and manage database systems effectively, and should be exposed for advancements in the field such as NoSQL and Big Data technologies.

Topics to be covered are:
  • Weeks 1-2: Introduction to database fundamentals, data modeling, and architecture.
  • Weeks 3-4: The relational model and design methodologies.
  • Weeks 5-6: Advanced SQL usage, data manipulation, and normalization techniques.
  • Weeks 7-11: Advanced topics including index structures, query optimization, transaction processing, NoSQL databases, Big Data, and Python integration.
  • Week 12: Information retrieval, web search, data mining concepts.
