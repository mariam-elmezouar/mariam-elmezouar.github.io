Auto_increment in MySQL

The AUTO_INCREMENT attribute in MySQL automatically assigns a unique identifier for new rows in a table:

  • Unique Identifier: Ensures each record has a distinct number for the PRIMARY KEY column.
  • Sequential Numbers: The value starts at 1 and increments by 1 for each new record, by default.
  • Inserts: When inserting data, you do not need to specify a value for the AUTO_INCREMENT column; MySQL handles it.

Example Usage:

CREATE TABLE students (
    student_id INT AUTO_INCREMENT,
    name VARCHAR(100),
    PRIMARY KEY (student_id)
);

INSERT INTO students (name) VALUES ('John Doe'); -- student_id will be 1
INSERT INTO students (name) VALUES ('Jane Smith'); -- student_id will be 2

ENUM Data Type in MySQL

ENUM is a string object that restricts data to a list of predefined values:

  • Predefined Values: The ENUM field can only contain one of the values listed during table creation.
  • Storage: Internally stored as integers, ENUM is space-efficient and fast.
  • Data Integrity: The field can only store a value that is listed in the column definition.

Example Usage:

CREATE TABLE shirt_orders (
    order_id INT AUTO_INCREMENT,
    shirt_size ENUM('XS', 'S', 'M', 'L', 'XL'),
    PRIMARY KEY (order_id)
);

INSERT INTO shirt_orders (shirt_size) VALUES ('M'); -- Accepts 'M' as a valid size
INSERT INTO shirt_orders (shirt_size) VALUES ('XXL'); -- Throws an error, 'XXL' is not in the list
