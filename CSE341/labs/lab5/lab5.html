Lab 5: Database Manipulation using MySQL

Objective

This laboratory is dedicated to practicing DML (Data Manipulation Language) requests in MySQL.

Task 1: Installation

  1. Install MySQL Workbench: If you haven’t already installed MySQL Workbench on your computer, download it from the official site.

    • MySQL Workbench Download

  2. Download Script: A script StudentDB.sql is provided to create the necessary database for practice.

Task 2: Practice DML in MySQL

  1. Load and Execute Script:
    • Open MySQL Workbench.
    • Go to the Edit SQL Script section.
    • Load the script StudentDB.sql.
    • Execute the script to create and populate the StudentDB database.

Database Schema

The database you will be working with has the following schema:

  • Student Table:

    • ID int
    • name varchar(20)
    • GPA real
    • sizeHS int

  • Campus Table:

    • location varchar(20)
    • enrollment int
    • camprank int

  • Apply Table:

    • ID int
    • location varchar(20)
    • major varchar(20)
    • decision varchar(20)

Task 3: SQL Queries

  1. Create Queries:

    • Based on the studentDB schema and data, create the queries as described in the file StudentDBQueries.sql.

  2. Test Queries:

    • Execute the file StudentDBQueries.sql in MySQL Workbench (Edit SQL Script section) to test your queries.
