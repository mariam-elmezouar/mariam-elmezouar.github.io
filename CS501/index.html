General Information

Topic this term: Data Analytics

This course will provide an introduction to the main topics in data analysis and knowledge discovery, including exploratory data analysis, frequent pattern mining, clustering, and classification

Learning Objectives

After taking this course, the students should be able to:

  • Describe the fundamental data mining tasks like pattern mining, classification, regression, and clustering.
  • Analyze the key algorithms of the main tasks.
  • Implement and apply the techniques to real world datasets.
