Lab 2

CSE451 Fall 2023

Introduction

In this lab, you will get introduced to the dataset you will be using for your project.

Objectives

  • Data familiarization: Read the dataset documentation to gain a clear understanding of its structure, content, and context.
  • Data manipulation: Perform basic data preprocessing tasks, including merging two datasets from two different periods of time, checking for missing values, outliers, checking the datatypes of columns, and identifying any challenges you think this data may pose in terms of analysis (how messy is it?).
  • Data Exploration: conduct exploration to uncover insights, patterns, and potential challenges within the dataset, using descriptive statistics, data visualization, and basic statistical tests.

Tasks

Task 1: Data Acquisition and Setup

  1. Download the dataset and associated documentation from here.
  2. Create a Jupyter Notebook for the lab.
  3. Load the dataset into the notebook using Pandas.

Task 2: Data Familiarization

  1. Read the dataset documentation thoroughly to understand its structure and context.
  2. Summarize the key information about the dataset, such as its source, and variables.

Task 3: Data Manipulation and Cleaning

  1. Identify missing values in the dataset appropriately. How prevalent are they? What are the possible techniques that can be used to handle missing values? and which do you think are appropriate in this case? We will discuss the techniques in more details in the next lecture.
  2. Detect outliers or anomalies if present. How can we address them, if present?
  3. Identify data types and check that they make sense. For example, is there a column that supposedly contains only numbers, and yet it has the datatype object?

Task 4: Exploratory Data Analysis (EDA)

  1. Generate descriptive statistics for key variables (e.g., mean, median, standard deviation).
  2. Create visualizations (e.g., histograms, scatter plots) to explore data distributions and relationships.
  3. Perform basic statistical tests, if relevant.
  4. Document any insights, patterns, or potential challenges observed during EDA.

Once you are happy with your understanding of the data, clean up your notebook and submit it to your instructor on Moodle. This is not graded. It is more for accoutability and for me to check that you are on the right track.
