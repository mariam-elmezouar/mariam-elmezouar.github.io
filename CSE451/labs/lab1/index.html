Introduction

In this lab, you'll dive into data science using Jupyter Notebooks. Data science combines programming and statistics to extract insights from data. This introductory lab focuses on data exploration and visualization, two key data science skills.

Objectives

  • Jupyter Notebook Basics: Get comfortable with Jupyter Notebooks by learning how to create, edit, and run code and Markdown cells.
  • Data Exploration: Learn to load datasets, examine data structure, identify missing values, and gain insights through basic summary statistics.
  • Data Visualization: Use Python libraries like Matplotlib and Seaborn to create essential visualizations like bar charts, scatter plots, and histograms.

Task 2: Basic Data Summary and Visualization

Objective: Explore the dataset in more depth through summary statistics and basic visualizations.

  • Identify which attributes (columns) are numerical and which are categorical.
  • In a new code cell, calculate basic summary statistics such as mean, median, and standard deviation for the numerical attributes.
  • Next, we will visualize some attributes of the dataset using simple plots. Two commonly used Python libraries used for visualizations are Matplotlib and Seaborn. In the initial code cell where pandas was imported, add the following imports: import matplotlib.pyplot as plt and import seaborn as sns

    • Matplotlib: a versatile Python library for creating various plots and charts, offering extensive customization options.

      • Highly customizable.
      • Supports a wide range of plot types.
      • Can be used with other libraries like NumPy and Pandas.
      • Ideal for creating publication-quality static plots.

    • Seaborn is a Python library built on Matplotlib, designed for creating visually appealing statistical graphics with ease.

      • Offers built-in themes and color palettes for aesthetics.
      • Provides high-level functions for complex plots.
      • Simplifies statistical visualization.
      • Integration with Pandas for seamless data handling.
  • Numerical attributes: using the code template below, construct a histogram to understand the distribution of some numerical attributes of your choice. Try to play around with the paramaters of the functions, and check if they has more parameters you can use.
