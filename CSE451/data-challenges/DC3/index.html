Data Challenge 3

CSE451 Fall 2023

This data challenge involves a synthetic dataset that has been generated to help you understand and practice data clustering techniques. The dataset contains two features for each data point. There is no meaning to the dataset features. There is no need to perform any data exploration.

Step 1: Load the Data

Begin by loading the dataset into your Jupyter Notebook environment.

Step 2: K-means Clustering

Apply the K-means clustering algorithm to the dataset. Start with an elbow analysis to determine the optimal number of clusters. Use a range of k values and plot the within-cluster sum of squares to find the elbow point.

  1. Perform the elbow method analysis.
  2. Choose the appropriate number of clusters based on your analysis.
  3. Apply K-means clustering with the determined number of clusters.
  4. Compute the number of elements in each cluster.
  5. Visualize the clustering using a scatter plot. Color the points based on the assigned cluster.

Step 3: Further Clustering with DBSCAN

Examine the scatter plot generated in the previous step. Identify any clusters that may not have been well-defined by K-means and could potentially be further refined with DBSCAN.

  1. Select one cluster that you think may benefit from further clustering with DBSCAN.
  2. Use the following parameters for DBSCAN: MinPts = 4 and Epsilon = 0.07.
  3. Apply DBSCAN to the selected cluster.
  4. Compute the number of clusters obtained and the number of elements in each cluster.
  5. Visualize the results using a scatter plot, showing the newly identified subclusters within the original K-means cluster.

Step 3: Discussion

What were your conclusions from this analysis and why might it be useful to perform clustering in two steps? Any problems that you think may not have been handled despite combining the two clustering approaches?

Submission

Once you have completed the above steps, submit your Jupyter Notebook with all cells executed and the visualizations displayed.
