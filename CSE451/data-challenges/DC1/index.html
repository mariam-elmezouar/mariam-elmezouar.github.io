Data Challenge 1

CSE451 Fall 2023

Task: Explore the ratings of FIFA players

Objective: provide a notebook report that explore and visualize the dataset from different perspectives. Document any insights gained from the exploration. Use a mix of statistical techniques and visualizations to achieve this objective.

  • Launch Jupyter Notebook and create a new textbook, and name it DG1_lastname.
  • Download the provided dataset. The dataset contains player ratings for the FIFA World Cup from 2006 to 2018. The columns of the dataset are described below:
    • Column Name Description Data Type
    player_url The URL of the player's FIFA profile. String
    short_name The player's short name. String
    long_name The player's long name. String
    age The player's age. Integer
    dob The player's date of birth. String
    height_cm The player's height in centimeters. Integer
    weight_kg The player's weight in kilograms. Integer
    nationality The player's nationality. String
    club The player's club. String
    overall The player's overall rating. Integer
    potential The player's potential rating. Integer
    value_eur The player's value in Euros. Integer
    wage_eur The player's wage in Euros. Integer
    player_positions The player's positions. String
    preferred_foot The player's preferred foot. String
    international_reputation The player's international reputation. Integer
    weak_foot The player's weak foot rating. Integer
    skill_moves The player's skill moves rating. Integer
    work_rate The player's work rate. String
    body_type The player's body type. String
    gk_positioning The player's goalkeeper positioning. Integer
    player_traits The player's traits. String
    attacking_crossing The player's crossing rating. Integer
    attacking_finishing The player's finishing rating. Integer
    attacking_heading_accuracy The player's heading accuracy rating. Integer
    attacking_short_passing The player's short passing rating. Integer
    attacking_volleys The player's volleys rating. Integer
    skill_dribbling The player's dribbling rating. Integer
    skill_curve The player's curve rating. Integer
    skill_fk_accuracy The player's free kick accuracy rating. Integer
    skill_long_passing The player's long passing rating. Integer
    skill_ball_control The player's ball control rating. Integer
    movement_acceleration The player's acceleration rating. Integer
    movement_sprint_speed The player's sprint speed rating. Integer
    movement_agility The player's agility rating. Integer
    movement_reactions The player's reactions rating. Integer
    movement_balance The player's balance rating. Integer
    power_shot_power The player's shot power rating. Integer
    power_jumping The player's jumping rating. Integer
    power_stamina The player's stamina rating. Integer
  • Once happy with your data exploration, clean up the notebook and add explanation/comments/insights as appropriate. Then, submit your notebook on Moodle.
