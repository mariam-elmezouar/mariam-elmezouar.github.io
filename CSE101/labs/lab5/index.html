Part 1: hullabaloo

Write a Python function hullabaloo that takes a character string as input and returns a shuffled version of the received string while ensuring no letters are missing. You are allowed to use only the random.randint function to shuffle characters.

Instructions:
  1. Check if the input string is empty or has only one letter. If it is, return the input string as is.
  2. Convert the input string into a list of characters. 
                # Example of how to convert the input string into a list of characters
            input_string = "example"
            char_list = list(input_string)
            print(char_list)  # Output: ['e', 'x', 'a', 'm', 'p', 'l', 'e']
  3. Shuffle the characters in the list using random.randint to create a complete shuffle. Do not simply swap two random characters; instead, ensure that all characters are randomly repositioned in the list.
  4. Continue shuffling until a different word is obtained compared to the input word.
  5. Return the shuffled word.
Example Usage:
input_word1 = 'plane'
shuffled_word1 = hullabaloo(input_word1)
print(shuffled_word1)

input_word2 = 'bonbon'
shuffled_word2 = hullabaloo(input_word2)
print(shuffled_word2)

Make sure to test the hullabaloo function with various words to ensure it works correctly and produces shuffled versions of the input words using only random.randint.

Part 2: Palindrome

A palindrome is a word, phrase, or sequence of characters that reads the same way forwards and backwards. For example, 'racecar', 'level', and 'deified' are palindromes.

Instructions:
  1. Write a Python function named is_palindrome that takes a string as input.
  2. Convert the input string to lowercase to make the palindrome check case-insensitive, using the lower() method.
  3. Create a new string containing only the alphabetic characters from the input string, by iterating through the characters using a loop and checking if each character is alphabetic using the isalpha() method.
  4. Check if the cleaned string reads the same forwards and backwards using loops. Iterate through the characters of the cleaned string to compare characters from the beginning and end.
  5. If the cleaned string is a palindrome, return True; otherwise, return False.
Example Usage:
input_str1 = 'racecar'
result1 = is_palindrome(input_str1)
print(result1)  # Output: True

input_str2 = 'Hello, World!'
result2 = is_palindrome(input_str2)
print(result2)  # Output: False

Make sure to test the is_palindrome function with various strings to check if they are palindromes or not.

Part 3: Simon

In the Simon game, you will create a program that plays Simon with the user. Simon is a memory game where there are four colored squares. The computer highlights the squares in a random order, starting with a series of length 1. The user's task is to click the squares in the same pattern that the computer showed them. The length of the series increases by 1 each time. The game ends when the user incorrectly repeats the pattern.

Instructions:
  1. Display four colored squares in a 2 by 2 pattern.
  2. Show the user a random sequence by highlighting the squares in order.
  3. Allow the user to click the squares to input the sequence that was shown to them.
  4. Inform the user of the length of the longest successful sequence.
