Part 1: Decimal to two's complement Binary

Create a Python function dec2bin_comp that converts an integer to its 8-bit two's complement binary string.

Constraints:

  • Do not use bitwise operators (&, |, ^, ~, <<, >>).
  • Do not use Python's built-in bin() function or any function that directly converts numbers to binary strings.
  • Do not use string functions like zfill, format, or libraries that handle binary numbers.
  • You must handle the conversion using arithmetic operations (addition, subtraction, multiplication, division) and control structures (loops, if statements).

Instructions:

  1. Define a function dec2bin_comp takes an integer value as its argument. - done
  2. Ensure value is within -128 to 127. If not, return an error message. - done
  3. Create a list of eight '0' characters for the binary digits. - done
  4. If value is non-negative, use steps seen in class for conversion.
  5. If value is negative, use steps seen in class for conversion.
  6. Join the binary digits list into a string and return it. - done
  7. Test the function with a range of inputs, including edge cases.

Test the program by calling your function with various valid and invalid numbers. Try a few positive and negative numbers (including limit values -129 (error), -128, -1 (all 1s), 0, +127, and +128 (error)) to make sure it's correct.

Part 2: Two's complement Binary to Decimal

Constraints:

  • Do not use Python's built-in int() function with a base argument to directly convert binary strings to integers.
  • Do not use any library functions or Python features that perform binary to decimal conversion automatically.
  • You must manually implement the two's complement conversion process using arithmetic and logic (loops, if statements, etc.).

Instructions:

  1. Define the function bin_comp2dec that takes a string bin_str as its argument.
  2. Verify that bin_str is exactly 8 characters long and contains only '0' or '1'. If not, return an error message.
  3. Check if the binary number is negative by looking at the first bit. If it is '1', the number is negative; otherwise, it's non-negative.
  4. For non-negative numbers, convert to decimal by iterating over the string and applying the binary to decimal conversion logic.
  5. For negative numbers, find the two's complement by converting to one's complement and adding 1, then convert to decimal and negate the result.
  6. Return the decimal value.
  7. Test the function with a range of binary inputs, ensuring both positive and negative values work as expected.
