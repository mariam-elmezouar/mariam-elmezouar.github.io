Lab 1: Simple calculations

CSE101 Fall 2023

Introduction

During this lab, you will write Python programs (scripts) that implement the requirements shown below. All work is to be done individually. If you receive assistance from anyone other than me, you must indicate this in your submission. Copying someone else's code or providing someone with your code to copy is not permitted.

Objectives

The objective of this lab is to give you practice with:

  • obtaining input values from the user,
  • performing simple calculations,
  • providing output to the user, and
  • performing simple looping operations.

Setup

Launch PyCharm and create a new project named lab1_lastname. For every task listed below, generate a new Python file within the project, using the name specified in the task.

1.   Celsius to Fahrenheit

The program, called c2f.py, asks the user for a temperature in Celsius and displays the equivalent temperature in Fahrenheit to two decimal places. Here is a sample run of the desired program: 

What is the Celsius temperature? 21
That is equivalent to 69.80 degrees Farenheit.

Hints: To format and display the result, you should use print. Read the documentation of the print function here, to figure out how to format the numbers to get two decimal places only. Include a new line character (\n) after the result to ensure that the next thing show in the console window starts on a new line.

2.   Fahrenheit to Celsius

The program, called f2c.py, performs the inverse of the c2f.m conversion and produces similar output.

3.   Average of 2 values

The program, called average2.py, asks the user to provide two values (one at a time) then displays the average of those values to three decimal places. Sample run: 

Enter two values to compute their average.
First value: 12
Second value: 13

The average is 12.500

4.   Average of several values

The program, called averagemany.py, asks the user how many values must be averaged, asks the user to provide each value in turn, then displays the average of those values to three decimal places. Sample run: 

How many values do you need to average? 3
Value 3: 10
Value 2: 20
Value 1: 50

The average is 26.667

Hint: You may use either a for loop or a while loop to ask the user for the required values. The for loop is actually simpler, but perhaps slightly more difficult to understand. Here is an example of a while that asks the user for input 3 times. 

count = 3

while count > 0:
    value = input(f"Value {count}: ")
    count = count - 1
