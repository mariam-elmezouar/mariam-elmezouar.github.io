Part 2: Teasure Hunt

Preamble: Accessing elements of a 2D vector using loops

The following code shows you how to access individual elements of a 1D vector and 2D vector using loops. Study the code very carefully (very important). On paper, draw the iterations of each loop, and examine how the variables change in every loop iteration.

 
      
        % Displaying elements of a simple vector
        vector_simple = [1 2 3];

        for element = 1:length(vector_simple)
          disp(vector_simple(element));
        end

        % Displaying elements of a 2D vector
        vector_2D  = [1 2 3; 4 5 6];

        for row = 1:size(vector_2D, 1)
          for col = 1:size(vector_2D, 2)
            disp(vector_2D(row, col));
          end
        end

Exercice 3

In a matlab function file called treasure.m, write a function treasure(x) that takes in a square array (i.e. nxn) that contains integers from 11 to 99. This array represents the treasure map. Your function must output a vector that contains all the positions (i.e. the path) that the pirate must visit in order to find the treasure. The method to follow the treasure map is described below :

  • The pirate always starts at position (1,1) on the map.;
  • Each position that the pirate visits contains an indication about which position the pirate is to visit next. Ex : if the position (1,1) contains the number 57 then the next position that the pirate must move to is (5,7);
  • The treasure is found when the position on the map and the number it contains are the same. Ex : if position (3,2) contains the number 32 then the treasure has been found.

Here is an example given the following map:

 
        
          map = [55 55 15 53 41
                 51 35 23 45 41
                 55 44 42 41 41
                 23 21 15 42 51
                 32 44 13 34 15]

When you execute path = treasure(map), then the function should return the vector [55,15,41,23]
