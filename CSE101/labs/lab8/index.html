Introduction

  • File input/output refers to reading data from (and writing data to) a file that is saved on a computer's external memory (e.g., hard drive, memory key, network drive, tape drive, etc.).
  • MATLAB provides a variety of functions for text file input and output.
  • The Matlab input functions: fgetl(), fgets, fscanf(), etc
  • The Matlab output functions: fprintf()
  • The Matlab file related functions: fopen(), fclose(), feof(), etc
  • Part of your task in this lab is to figure out how to use these functions to achieve the desired results.

There are 3 basic steps to accessing data in a file:

  1. Open the file
  2. Read data from the file OR write data to the file (but typically not both)
  3. Close the file

To open a file

  • Use File_ID = fopen(File_name, Access_mode) function.
    • The first argument to fopen() is the name of the file.
    • The second argument to fopen() is a string that specifies the type of access that is desired to the file. Some of the options are:
      • 'r' open an existing file for reading only (default permission)
      • 'r+' open an existing file for reading and writing
      • 'w' delete/create a file and open it for writing
      • 'a' open/create a file for appending (writing data after the existing contents of the file)

To read (or write) data to a file

  • To read data from a file: fgetl(), fgets, fscanf(), etc
  • To write data to a file: fprintf()

Part 1: Nonsensical Paragraph Generator

We will write a program that generates a random essay using words that are found in another piece of text. The text that we'll use is found at this url: https://www.cs.queensu.ca/home/cords2/cs.txt

To read and store text from a url, use the following code:

      
        url = " https://www.cs.queensu.ca/home/cords2/cs.txt";
        code = webread(url);
        str = extractHTMLText(code);

This text was taken from an article written by Joy Buolamwini published in TIME on February 7, 2019. Buolamwini is a computer scientist, founder of the Algorithmic Justice League and a poet of code.

Your program will read the text from the website, generate a dictionary of words sorted by the first letter of the word. Your program will then use this dictionary to generate paragraphs of text to form an essay using randomly generated choices for the words.

Here is an outline of the functions that are required to achieve the goal, their inputs and what they should return. Your program must follow this outline and not deviate. Deviations from the outline (that is, changing the inputs or outputs, adding functions, leaving out functions etc) will result in a lower mark.

  • wordList = readWebPage(url) - this function takes one input, the URL from which to read the text, and it returns a vector where each element is a word in the text found on the website. The function should read the contents of the website into a string and split the string to a list of words (hint: use the function split()).
  • cleanedWordList = cleanDictionary(wordList) - this function takes as a parameter a list of words and modifies the words in the list to remove punctuation marks that appear at the end of some of the words. You should check the last character against all types of punctuation, not just periods or commas.
  • sortedWordList = sortDictionary(cleanedWordList) - this function takes as a parameter a list of cleaned words and sorts the words in the list in alphabetical order.
  • paragraph = makeParagraph(sortedWordList, numberofWords) – this function takes the dictionary of words and creates a paragraph (a string) consisting of numberofWords words. The choice of the words is random . The paragraph is returned.
  • text = createEssay(sortedWordList) - in this function you will ask the user how many paragraphs they want to generate. For each paragraph, you will ask the user how many words should be in that paragraph. To generate each paragraph, you will call makeParagraph() and add the string that is returned to the "essay". Be sure to add a new line "\n" after each paragraph so that it will appear on a new line when you write it to a file.
  • writeToFile(text) – this function takes a string as a parameter. It opens a file called "essay.txt" and writes the text string to the file. Be sure to check that the file was opened properly.
  • The functions will be called in this order with the proper inputs:
    readWebPage()
    cleanDictionary()
    sortDictionary
    createEssay() -- createParagrapgh will be called inside createEssay()
    writeToFile()
