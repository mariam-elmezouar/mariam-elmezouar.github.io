Part 1: Building the Encrypt and Decrypt Functions

This exercise builds upon the simplified Caesar cipher discussed in class. You will enhance your understanding by developing a more comprehensive version of the cipher.

Task Description

Write a script to decode a message and display it to the screen. Your task is to write two Python functions:

  1. Encrypt Function: This function takes a string as input and shifts each character in the string by a fixed number of steps in the ASCII table.
  2. Decrypt Function: This function reverses the process, taking the encrypted string and the same number of steps to return the original message.

Requirements

  • The shifting should wrap around the ASCII table. For example, if you shift ‘z’ by 1, it should become ‘a’.
  • Only shift alphabetic characters (a-z and A-Z). Non-alphabetic characters (like spaces, punctuation) should remain unchanged.
  • The number of steps to shift should be a parameter to your functions.

Example

If the input string is “Hello, World!” and the shift is 3, the encrypted string should be “Khoor, Zruog!”.

Hints

  • Use the ord() function to convert a character to its ASCII code and chr() to convert an ASCII code back to a character.
  • Check whether a character is uppercase or lowercase, as they have different ranges in the ASCII table.
  • Be careful with characters near the end of the alphabetic range in the ASCII table; you need to wrap around to the beginning of the alphabet.

Part 2: File Encryption and Decryption

Objectives

  • Use the encrypt and decrypt functions from part 1 to encrypt and decrypt the contents of a file.

Steps

  1. Read the contents of a file named input.txt (provided).
  2. Decrypt the contents using the decrypt function using a shift = 5.
  3. Write the decrypted contents to a file named decrypted.txt.
  4. Read the contents of decrypted.txt.
  5. Encrypt back the contents using the encrypt function and a shift = 3 this time.
  6. Write the encrypted contents back to a file named encrypted.txt.
