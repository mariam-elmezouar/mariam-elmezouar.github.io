Lab 7: Hangman and Treasure hunt

CSE101 Fall 2023

Introduction

During this lab, you will implement the game hangman and a treasure hunt.

Requirements

Carefully read and follow the instructions. Points will be deducted for working solutions that do not follow the requirements.

Your code needs to follow good coding habits, including proper commenting.

At the top of your file, clearly indicate your name and student number.

Getting Setup

Please follow these setup instructions.

In PyCharm, create a new project, download, and add the following files to it:

Files to Download:
  1. my_converter.py: This file contains the implementation of the Dec2BinComp and BinComp2Dec functions. You will need to complete the implementation of these functions in this file.
  2. test_my_converter.py: This file contains test cases for the Dec2BinComp and BinComp2Dec functions. You can use these test cases to verify the correctness of your implementations.

Once you have downloaded these files, you can start working on your lab. Feel free to add more test cases to test_my_converter.py to ensure your functions work correctly.
