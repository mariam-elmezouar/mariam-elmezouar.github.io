Lab 2: Selection and Repetition

CSE101 Fall 2022

Introduction

During this lab, you will get to write longer and more complex MATLAB solutions that combine the use of variables, operations, selection and repetitions, as needed.

Objectives

The objective of this lab is:

  • To solidify the concepts of selection and repetition in MATLAB
  • To practice programming within the Matlab IDE
  • To test your code

Problem 1

Create a new matlab file, name it lastname_prob1.m. Write the solution to the problem below in the file you have just created.

Write MATLAB code to solve the quadratic equation using the formula:

Your code must:

  • Verify if there will be a division by zero. If so, do not perform the division, but give the user an appropriate message informing them of the problem;
  • Verify if the root will be real. If it is imaginary then give appropriate message to inform the user;
  • If the roots are real then display both values of x.

Note: Make sure that you test your code with various values of a,b, and c to ensure that you have tested each possible case of output.

Problem 2

Given the vector x = [5 3 5 0 1 4], write code that will:

  • Add up the values of the elements (without using the Matlab function sum), and displays the total
  • Computes and displays the running sum. For element i, the running sum is the sum of the elements from 1 to i, inclusive (do not use the Matlab function cumsum).

Problem 3

Create a new matlab file, name it lastname_prob2.m. Write your solution to the problem below in the file you just created.

You have a mortgage of $300,000. At the end of each month the bank will charge you 0.5% interest on the remaining mortgage. At the end of each month, you will pay a fixed amount of $1800 to reduce the mortgage.

Your code must answer the following questions:

  • In how many years will you be done paying for your house?
  • How many payments exactly will you have made?
  • How much will you have paid in interest during these years?

Note: make sure that the output of your program is clear and informative by displaying appropriate messages.
