Problem 1: Mirror Swap String

Write a Python function named mirror_swap_string that takes a string as input and returns a new string where the characters are swapped in a mirrored fashion. The first character should be swapped with the last, the second with the second to last, and so on, until all characters have been swapped.

Problem 2: String Symmetry Challenge

Write a Python function named string_symmetry_challenge that performs a series of manipulations on a given string. The function should reverse the string, concatenate the original string with this reversed string, find the middle of the concatenated string, slice the second half, repeat this sliced half twice, and concatenate this repeated string with the first half of the original string.

Problem 3: Tracing Sorting Algorithms

Analyze and write out the sorting trace for Bubble Sort, Selection Sort, and Insertion Sort using the list [6, 3, 8, 5, 2]. Trace the steps each algorithm takes to sort the list, writing out the list as it appears after each significant action in the algorithm.

Problem 4: Number Decomposition

Write a Python function named number_decomposition that uses a while loop to break down a given positive integer into its individual digits and calculates their sum. The function should take a single integer as input, and returns the sum of the digits that compose the integer.

Example:

  • Input: 1234
  • Expected Output: 10 (since 1 + 2 + 3 + 4 = 10)

Problem 5: Countdown String Creator

Write a Python function named countdown_string_creator that constructs a countdown string from a given number using a while loop.

Instructions:

  1. The function should accept a single integer as input.
  2. Using a while loop, decrement this number every iteration and concatenate its string representation to a growing string.
  3. Continue concatenating until the number reaches zero.
  4. The function should return the final concatenated string, which includes all numbers from the input down to zero.

Example:

  • Input: 5
  • Expected Output: '543210'
