Python Tuples and Sets

Tuples

  • Definition: A tuple is an immutable (unchangeable) sequence in Python.
  • Creation: Use parentheses () .
  • Immutable: Once created, items cannot be added, removed, or changed.
  • Ordered: Items have a defined order, and that order will not change.
  • Indexing: Support indexing and slicing like lists.

Example of Tuple

# Creating a tuple
my_tuple = (1, 2, 3)
print(my_tuple)  # Output: (1, 2, 3)

# Accessing elements
print(my_tuple[1])  # Output: 2

# Slicing
print(my_tuple[0:2])  # Output: (1, 2)

# Tuples are immutable
# my_tuple[1] = 5  # This will raise an error

Sets

  • Definition: A set is an unordered collection of unique items in Python.
  • Creation: Use curly braces {} .
  • Unordered: Items do not have a defined order, and their order can change.
  • Mutable: You can add or remove items from a set.
  • No Duplicates: Automatically removes duplicate items.

Example of Set


# Creating a set
my_set = {1, 2, 3}
print(my_set)  # Output: {1, 2, 3}

# Adding elements
my_set.add(4)
print(my_set)  # Output: {1, 2, 3, 4}

# Removing elements
my_set.remove(2)
print(my_set)  # Output: {1, 3, 4}

# Sets automatically ignore duplicates
my_set = {1, 2, 2, 3, 3, 3}
print(my_set)  # Output: {1, 2, 3}

# Accessing individual items in sets using a loop
for item in my_set:
    print(item)  # Prints each item in the set

# Converting sets to lists to access individual items using indices
my_list = list(my_set)
print(my_list[0])  # Accesses the first item in the set, now a list

Differences Between Lists, Tuples, and Sets

  • Lists:

    • Mutable (can add, remove, or change items).
    • Ordered (items have a defined order).
    • Indexed (can be accessed by an index).
    • Can have duplicate values.

  • Tuples:

    • Immutable (cannot change items once created).
    • Ordered and indexed.
    • Can have duplicate values.

  • Sets:

    • Mutable (can add or remove items).
    • Unordered (items do not have a fixed position).
    • Cannot be indexed or sliced.
    • Cannot have duplicate values. Automatically removes duplicates.
