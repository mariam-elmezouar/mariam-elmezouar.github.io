File Access in Python Tutorial

Working with files is a foundational skill in programming. Python simplifies file operations with a set of built-in functions. This tutorial covers how to read from and write to files.

Opening a File

To open a file, you use the open() function. It requires the path to the file and an optional mode parameter.

file = open('example.txt', 'r')  # Open for reading (default mode)
file = open('example.txt', 'w')  # Open for writing, if file already exists, this will overwrite old content

Reading from a File

Once a file is opened, you can read its contents in several ways:

  • read(): Reads the entire file.
  • readline(): Reads the next line from the file.
  • readlines(): Reads all the lines into a list.
content = file.read()  # Reads the whole file
line = file.readline()  # Reads the next line
lines = file.readlines()  # Reads all lines into a list

Writing to a File

To write to a file, open it in write 'w' or append 'a' mode and use write() or writelines().

file = open('example.txt', 'w')
file.write('Hello, World!')  # Writes a string to the file
file.writelines(['Hello', 'World'])  # Writes a list of strings to the file

Closing a File

After you're done with a file, close it using close(). It's a good practice to release resources.

file.close()

The with Statement

The with statement simplifies file handling by closing the file automatically after the with block.

with open('example.txt', 'r') as file:
    content = file.read()
# The file is automatically closed after the with block

Example: Reading a File

Here’s how to open a file, read from it, and then close it:

with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

Example: Writing to a File

To write to a file, ensuring that it’s properly closed afterward:

with open('example.txt', 'w') as file:
    file.write('Goodbye, World!')

Handling File Errors

To handle errors, such as the file not existing, use what is called a try-except block:

try:
    with open('nonexistent.txt', 'r') as file:
        content = file.read()
except FileNotFoundError: # this detects that the file was not found
    print('The file does not exist.')
except IOError as e: # this detects that there was some other error opening the file
    print(f'An error occurred: {e.strerror}')

Complete Example: Processing File Contents

Imagine we have a text file containing names, and we want to create a new file that contains each name with a greeting. The original file (names.txt) looks like this:

Alice
Bob
Charlie

We want to read this file, add a greeting to each name, and save the results to a new file (greetings.txt).

Step 1: Read the Original File

# Open the original file and read the lines
with open('names.txt', 'r') as file:
    names = file.readlines()

Step 2: Manipulate the Contents

We’ll strip any whitespace and add a greeting to each name:

# Strip whitespace and add a greeting to each name
greetings = []
for name in names:
    greetings.append(f'Hello, {name.strip()}!\n')

Step 3: Write to a New File

Now, we write the manipulated contents (greetings) to a new file:

# Write the greetings to a new file
with open('greetings.txt', 'w') as file:
    file.writelines(greetings)

The greetings.txt file will now contain:

Hello, Alice!
Hello, Bob!
Hello, Charlie!

Bonus: Overwriting the Original File

If you wanted to overwrite the original file instead of creating a new one, you could open it in write mode after processing the content and write the manipulated lines back to it.

# Open the original file in write mode and overwrite it
with open('names.txt', 'w') as file:
    file.writelines(greetings)

Error Handling

In practical applications, it's essential to handle potential errors. Here's how to manage exceptions when working with file operations:

try:
    with open('names.txt', 'r') as file:
        names = file.readlines()
    greetings = []
    for name in names:
        greetings.append(f'Hello, {name.strip()}!\n')
with open('greetings.txt', 'w') as file:
    file.writelines(greetings)
except FileNotFoundError:
    print('The file does not exist.')
except IOError as e:
    print(f'An error occurred: {e.strerror}')
