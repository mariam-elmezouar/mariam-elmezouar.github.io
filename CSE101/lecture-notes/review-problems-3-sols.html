Solution to Problem 1: Mirror Swap String

def mirror_swap_string(s):
    # Convert the string into a list of characters for easy manipulation
    char_list = list(s)

    # Use half the length of the string for the range to avoid double swapping
    for i in range(len(char_list) // 2):
        # Swap the ith character with the character at the mirrored position
        temp = char_list[i]
        char_list[i] = char_list[-i - 1]
        char_list[-i - 1] = temp

    # Convert the list of characters back to a string
    return ''.join(char_list)

Solution to Problem 3: Tracing Sorting Algorithms

Bubble Sort Trace:

  • Start: [6, 3, 8, 5, 2]
  • Pass 1: Compare 6 and 3, swap: [3, 6, 8, 5, 2]
  • Pass 2: Compare 6 and 8, no swap: [3, 6, 8, 5, 2]
  • Pass 3: Compare 8 and 5, swap: [3, 6, 5, 8, 2]
  • Pass 4: Compare 8 and 2, swap: [3, 6, 5, 2, 8]
  • Result after Pass 1: [3, 6, 5, 2, 8]
  • Continue similar passes until fully sorted: [2, 3, 5, 6, 8]

Selection Sort Trace:

  • Start: [6, 3, 8, 5, 2]
  • Pass 1: Select minimum (2), swap with first element: [2, 3, 8, 5, 6]
  • Pass 2: Select next minimum (3), already in place: [2, 3, 8, 5, 6]
  • Pass 3: Select next minimum (5), swap with third element: [2, 3, 5, 8, 6]
  • Pass 4: Select next minimum (6), swap with fourth element: [2, 3, 5, 6, 8]
  • Fully sorted: [2, 3, 5, 6, 8]

Insertion Sort Trace:

  • Start: [6, 3, 8, 5, 2]
  • Pass 1: Insert 3 before 6: [3, 6, 8, 5, 2]
  • Pass 2: 8 is already in the correct position: [3, 6, 8, 5, 2]
  • Pass 3: Insert 5 before 8: [3, 6, 5, 8, 2]
  • Pass 4: Insert 2 at the beginning: [2, 3, 6, 5, 8]
  • Continue inserting each element in its correct position: [2, 3, 5, 6, 8]

Solution to Problem 4: Number Decomposition

def number_decomposition(n):
    sum_of_digits = 0
    while n > 0:
        sum_of_digits += n % 10
        n //= 10
    return sum_of_digits

Solution to Problem 5: Countdown String Creator

def countdown_string_creator(number):
    countdown_string = ""
    while number >= 0:
        countdown_string += str(number)
        number -= 1
    return countdown_string
