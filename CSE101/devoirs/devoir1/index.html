Ligne de commande Linux

Tout au long de ce cours, vous utiliserez largement la ligne de commande Linux. Afin de vous aider à vous familiariser avec l'utilisation de Linux, vous suivrez quelques tutos assez simples (mais importants). Vous avez deux choix:

Option 1 (Recommendé)

Cette option nécessite l'utilisation de l'une de vos machines virtuelles Linux dans l'environnement de laboratoire (ou en créer une sur votre PC personnel). C'est très instructif.

  • Accédez au tuto Ryan’s Tutorials.
  • Complètez les secitons suivantes:
    • The Command Line
    • Basic Navigation
    • More About Files
    • Manual Pages
    • File Manipulation

Option 2

Cette option tire parti d'un tuto gratuit de Linux en ligne qui comprend un simulateur Linux rudimentaire, mais le contenu est assez minimal.

  • Accédez au tutoriel sur Linux Survival.
  • Compléter les trois premiers modules.
