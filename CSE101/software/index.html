The GoGoGraphics package

A simple graphics package for Python, primarily intended for use in teaching an introductory programming course. It was developped by Dr. Greg Phillips from the Electrical and Computer engineering at the Royal Military College. You can:

  • Download the complete GoGoGraphics package, which includes the Python source files plus HTML and PDF documentation;
  • Browse the HTML documentation online, or
  • Get a PDF printable version of the documentation.

Instructions on how to use and explore the package

First, watch this quick demo. Then, read the documentation to gain a better understanding of what the package has to offer.

To get comfortable with the GoGoGraphics package, create a new Python script, import the package, and try creating and customizing shapes. You can also refer to the examples and tests in the respective folders for guidance.

The package contains the following folders:

  • doc:

    This folder contains documentation for the GoGoGraphics package. Documentation describes the different functions that the package offers with some examples.

  • examples:

    This folder is where you can find example code that you can run to demonstrate how to use the GoGoGraphics package.

  • gogographics.py :

    This is the main Python module or script for the GoGoGraphics package. It contains the package's functions that users can import and use in their own Python code *** DO NOT MODIFY ***

  • tests :

    This folder contains test scripts that are used to verify the correctness and functionality of the GoGoGraphics package.
