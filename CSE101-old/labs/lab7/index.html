Lab 7: Numbering Systems

CSE101 Fall 2022

Introduction

During this lab, you will implement some of the base conversions discussed in class.

Objectives

The objective of this lab is:

  • To use MATLAB to solve more complex problems, involving base conversations.
  • To practice programming within the Matlab IDE
  • To test your code

Getting setup

The following starting code contains three files: two Matlab function files and one test file . Unzip and save the folder somewhere safe. Keep all files together in the same folder. Open the three files in Matlab, and try running test.m. Nothing should happen at this point since the functions are not implemented yet. The purpose of the test files is to test the two functions you will be implementing in this lab.

Few rules to note about this lab:

  • This lab is individual.
  • At the top of each function file, add a comment as follows %Name: X
  • Part of your grade on this lab will be based on the documentation of your code, i.e. adding comments that explain your logic and what your code is doing. Add comments as needed and as clearly as possible.
  • Use proper indentation and meaningful variable names.

Part 1: Decimal to two's complement Binary

Write a Matlab function Dec2BinComp which takes as input an integer between -128 and +127 and returns the equivalent 8-bit two's complement binary in the character array valbin

WARNING: Your code must first verify that the integer is between -128 and +127. If so, it should report an error with the error command and a clear message about the error such as error('Value out of range'). The error command automatically terminates the function.

Test the program by calling your function with various valid and invalid numbers. Try a few positive and negative numbers (including limit values -129 (error), -128, -1 (all 1s), 0, +127, and +128 (error)) to make sure it's correct.

Part 2: Two's complement Binary to Decimal

Write a Matlab function BinComp2Dec which takes as input a two's complement 8-bit signed binary (i.e. a binary that can be possibly positive or negative) stored in the character array valbin and returns the equivalent decimal in val.

Your solution must contain a for loop.

WARNING: Your code must first check that the string is indeed 8 characters long. It must also check that it only contains 0s and 1s. If not, it must report an error with the error command and a clear message about the error. There must therefore be at least two different error messages, one to indicate that the length of the string is invalid error('Invalid length'), and one to indicate that a character of the string is invalid error(' Invalid character'). The error command automatically terminates the function.

For example, your function can start with

 
    %verifying length of array 
      if length(valbin) ~= 8
           error('invalid length'); 
      end
