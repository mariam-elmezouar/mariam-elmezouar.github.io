Lab 3: functions

CSE101 Fall 2022

Introduction

During this lab, you will practice the use of functions to write modular and reusable programs.

Objectives

The objective of this lab is:

  • To practice the concept of functions in MATLAB
  • To solidify the concepts of selection and repetition in MATLAB
  • To practice programming within the Matlab IDE
  • To test your code

Part 1

To start, we will practice with simple functions. The first function is mysum that you save in a file called mysum.m. This function takes two inputs a and b and returns their sum.

function c = mysum(a, b)

Next, write the function multiply_with_sum in the same file. This function takes two inputs a and b. Then, using a loop, it returns the multiplication of a and b using the sum as explained below.

a * b = a + a + .... + a (b times)

An incomplete implementation of this function is provided below:

 
    function c = multiply_with_sum(a, b)
          c = 0
          for i = [1:b]
              % complete this line
          end
      end

At first, you can use the symbole + to complete the missing line above. Once you get your code to work properly, you can instead call your function mysum to perform the addition. Test your code thoroughly by calling the functions with different values.

At this point, call your instructor to see the final program. You do not need to submit Part 1.

Part 2

You matlab file in Part 2 needs to be a function, i.e., it needs to start with the following declarative line: 
 
      function v2 = inverse_lastname(v1)
        %This function will inverse the content of the vector v1

where lastname is your last name. v1 is the input of the function and v2 is the output of the function. v2 should have the same elements as v1 in reverse order. The name of the Matlab file shoule be the same as the name of the function, i.e. inverse_lastname.m. Tip: use a second vector to place the elements gradually as they are being reversed, and then return that vector.

To find out the length of a vector v, Use the Matlab function length(v). To access the i-th element of a vector v, use the notation v(i).

Test the program by calling your function with various vectors and verifying that they are inverted. Your program must work correctly in all cases (empty vector, vector with only 1 element, vector with 2 elements, etc.).

Your program should follow good coding conventions (use of comments, indentation, and meaningful variable names).

General tips

  • Read the instructions several times if necessary, analyze, fully understand what is required.
  • Take notes, make diagrams to visualize your ideas if necessary.
  • Solve by hand, on paper: these will be the general steps of the algorithm.
  • Avoid details initially. Add details in later refinement iterations of the algorithm.
  • Refine each step.
  • Test the algorithm by testing with small vectors, even empty ones.
  • Once the algorithm is completed to your liking, translate it into Matlab
