Lab 6: Strings

CSE101 Fall 2022

Introduction

During this lab, you will gain a deeper understanding of string manipulation in Matlab.

Objectives

The objective of this lab is:

  • To learn how to manipulate strings.
  • To use MATLAB to solve more complex problems, involving the string functions.
  • To practice programming within the Matlab IDE
  • To test your code

Getting setup

The following starting code contains four files: three Matlab function files and one test file . Keep all files together in the same folder. Open the four files in Matlab, and try running test.m. Nothing should happen at this point since the functions are not implemented yet. The purpose of the test files is to test the three functions you will be implementing in this lab.

Few rules to note about this lab:

  • You can work in teams of up to three students. All members should contribute equally. If not, the members will not get the same grade on this lab.
  • At the top of each function file, add a comment as follows %Team members: X, Y, and Z.
  • Part of your grade on this lab will be based on the documentation of your code, i.e. adding comments that explain your logic and what your code is doing. Add comments as needed and as clearly as possible.
  • Use proper indentation and meaningful variable names.

Part 1: hullabaloo

Write a Matlab function hullabaloo which returns a shuffled version of the received character string. For example, if the word received as input is 'plane' the returned string could be 'naple' or 'enapl' or even 'enpla' etc. Attention: the returned word must contain exactly the same letters, only in a shuffled order! The shuffled word should never be the same as the initial word, except of course, if the initial word has only one letter.

You can access a string as if it were an array. So length(s1) returns the length of the word, s1(1) the first letter, s1(n), the nth letter, etc. You must use the rand function which generates a random real number between 0 and 1. Use help rand to see the details. The function fix which removes the digits after the decimal point (you can use round) will also be useful to you. For example, if we want an integer between 1 and 10, we can use this code:

fix(rand()*10)+1

Test the program by calling your function with various words and checking that no letters are missing. Your program should work correctly in all cases, even if the string is empty, and even if there are repeated letters in the word. For example, for the word 'bonbon', 'nbonbn' would not be a good answer. Your program should follow coding conventions (comments, proper indentation, and representative variable names).

Part 2: Palindrome

A palindrome is a word, or group of words, that reads the same way forwards and backwards. For example: 'laval', 'rotor' 'radar', and 'aabbcbbaa' are palindromes. Write a Matlab function that receives a string as input argument and returns true if the string is a palindrome and false if it is not.

Part 3: Largest common substring

Write a function that receives two strings and finds the largest common substring. Use the strcmp and strfind functions from Matlab. For example, if you receive s1 = 'abcabbaccb' and s2 = 'abcba', the largest common substring is 'abc'. Be careful, this problem will require that you think carefully about your algorithm before coding it. If the shortest string is 8 characters, you must first find out if it is included in the longest. Otherwise you have to try the 2 substrings of 7 characters long, otherwise the 3 substrings of 6 characters long, etc, until you find a match. Otherwise, return an empty string ''.

Test your algorithm on the following strings:

‘abcabbaccb’ and ‘abcba’ (answer ‘abc’)
‘adhfkuhakjhfkjsdghkjsa’ and ‘hakjhfddhajskh’ (answer ‘hakjhf’)
‘askjdhfysadghfjkahdfgfg’ and ‘jkahdfgftyuewh’ (answer ‘jkahdfgf’)
'lahfdusadfilrug' and 'lrugjhflyhewarfhewi' (answer 'lrug')
'ladshuifuik' and 'fgakgfaifuiahfuiehw' (answer 'ifui')
‘afhgaehflrghrulgfla’ and ‘ndegfghrulgfelhfkw’ (answer ‘ghrulgf’)
‘askfghfu’ and ‘hfiasdhkaskf’ (answer ‘askf’)
