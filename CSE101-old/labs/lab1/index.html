Lab 1: Understanding variables and operations

CSE101 Fall 2022

Introduction

During this lab, you will get more familiar using Matlab arrays and applying operations on them.

Objectives

The objective of this lab is:

  • To get familiar with arrays
  • To practice programming within the Matlab IDE
  • To test your code

Section 1

In this task, you will learn how to create arrays using different approaches, and how to manipulate arrays.

  • Log into your MathWorks account
  • Complete the next three modules from the online training that was started in the previous lab

The tasks to be completed are Vectors and Matrices, Indexing into and Modifying Arrays, and Array Calculations.

You do not need to show the completion of your tasks to the instructor. What you have learned will be used in the following section. You can move on to the next section.

Section 2

Open the Matlab editor. For each of the following exercises, open a new .m file and save it with the name lastname_exerciseX.m, where X refers to the exercise number. Make sure you know where the Matlab files are saved. You will need to submit them to your instructor.

Exercise 1

Create a row vector of the even integers between 10 and 20 inclusive.

Exercise 2

Let x = [5 1 3 7] and y = [5 2 1 3] (Note that x and y are row vectors)

  • Raise each element of x to the power specified by the corresponding element in y
  • Divide each element of y by the corresponding element in x
  • Multiply each element in x by the corresponding element in y, calling the result "z"
  • Add up the elements in z and assign the result to a variable called "w"
Exercise 3

Given the array A = [ 2 4 1 ; 6 7 2 ; 3 5 9], provide the commands needed to

  • assign the first row of A to a vector called x1
  • assign the last 2 rows of A to an array called y
  • compute the sum over the columns of A
  • compute the sum over the rows of A
Exercise 4

Given that x = [1 5 2 8 9 0 1] and y = [5 2 2 6 0 0 2], execute and explain the results (as comments in the code using the symbol %) of the following commands:

  • x > y
  • y < x
  • x == y
  • x <= y
  • y >= x
  • x | y
  • x & y
  • x & (~y)

Submission

Prepare a zipped file (.zip) of your four matlab files, and name it lastname_lab1.zip. Submit the zipped file in Moodle by Monday September 19th at 11:59pm.
